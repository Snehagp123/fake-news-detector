# 📰 Fake News Detection System

This is a Machine Learning project that detects whether a news article is real or fake.

## 🔧 Technologies Used
- Python
- Scikit-learn
- NLP (TF-IDF)
- Streamlit

## 🚀 Features
- Detect fake vs real news
- Confidence score
- Interactive UI

## ▶️ Run the project
pip install -r requirements.txt
streamlit run app.py
